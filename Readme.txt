Forum-App

This is a web application which is a question discussion forum. 

React version: 0.1.0
Node.js:v.14.15.1
npm: 6.14.8
mongod: v4.4.1

To start the MongoDB database:
1. go to the directory where mongodb is installed
  mongod --dbpath "PATH/backend/data"
2. go to the directory where mongodb is installed
  mongo
  use forum
3. cd backend
  nodemon server


To start web server:
cd projectapp
npm start

To access the main page:
http://localhost:3000/main
